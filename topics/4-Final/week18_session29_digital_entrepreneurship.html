Lean Startup Methodology

BUILD → MEASURE → LEARN

by Eric Ries

The Cycle

  • Build: Create MVP (minimum viable product)
  • Measure: Put in front of real users, track results
  • Learn: What did data tell you?
  • Decide: Pivot or persevere?

Key Principles

  • Validated learning – use data, not assumptions
  • Pivot or persevere – change if data says so
  • MVP first – simplest version to test
  • Innovation accounting – actionable metrics

Don't build 2 years then discover
nobody wants it.

Build small. Test fast. Learn quick.
