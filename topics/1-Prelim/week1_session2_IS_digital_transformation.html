05Part 1

What Is an
Information System?

06Definition

Information System

IS = organized combination of people, hardware, software, data, and procedures that collects, processes, stores, and distributes information for decision-making.

07Formula

5 Components

People+Hardware+Software+Data+Procedures=IS
08People

Component 1: People

Most important and most overlooked. End users, IT staff, managers, data entry operators.

09Hardware

Component 2: Hardware

CategoryExamples
ProcessingServers, PCs, phones
StorageHDD, SSD, cloud
Input/OutputKeyboards, scanners, monitors
NetworkRouters, switches
10Software

Component 3: Software

TypeExamples
SystemWindows, Linux
ApplicationERP, CRM, LMS
DatabaseMySQL, MongoDB
11Data

Component 4: Data

TypeExample
TransactionalPurchases, enrollments
MasterStudent profiles, catalogs
AnalyticalTrends, distributions
Real-timeSensor readings, prices
12Procedures

Component 5: Procedures

  • Operating – daily use
  • Backup – protect data
  • Security – access, passwords
  • Recovery – when things fail
  • Training – how users learn
13Key

Remove ONE component = system fails.
All five work together.

21Part 3

Three D's

22Level 1

Digitization

Digitization = analog → digital format. No process change.

23Level 2

Digitalization

Digitalization = digital tools improve process. Same business model.

24Level 3

Digital Transformation

DX = redesign business models, culture, operations. Changes everything.
